# 🚨 Telegram Mass Reporter Bot

## Features
- ✅ Unlimited String Session support
- ✅ Target management (User/Channel/Group)
- ✅ Auto-join channels before reporting
- ✅ Multiple reporting reasons
- ✅ Rate limiting (10s delay)
- ✅ Real-time statistics
- ✅ SQLite database
- ✅ Inline keyboard interface
